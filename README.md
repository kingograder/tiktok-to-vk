# tiktok-to-vk

Синхронизация коллекций TikTok с VK Clips.

Автоматически находит посты в коллекции TikTok, скачивает видео, конвертирует горизонтальный контент в вертикальный формат (1080x1920) и загружает их как VK Clips. Ведёт учёт обработанных постов в локальной SQLite базе данных.

## Возможности

- **Обход коллекций** -- получает все посты из коллекции TikTok через внутренний API
- **Скачивание видео** -- загрузка через yt-dlp с имитацией браузера
- **Вертикальная конвертация** -- вписывает или поворачивает горизонтальные видео до 1080x1920
- **Загрузка в VK** -- загрузка клипов через VK API с ограничением частоты и опросом статуса
- **SQLite учёт** -- запоминает, какие посты уже скачаны и загружены
- **Режим демона** -- работает в фоне с настраиваемым интервалом опроса
- **Корректное завершение** -- обрабатывает SIGINT/SIGTERM, до завершает текущий цикл
- **Множественные коллекций** -- поддержка нескольких коллекций через запятую в `TIKTOK_COLLECTION_URL`

## Требования

- Python 3.10+
- ffmpeg (для обработки видео)
- Куки сессии TikTok (`cookies.txt` в формате Netscape)
- Токен VK с правом загрузки видео

## Получение VK токена

1. Откройте [vkhost.github.io](https://vkhost.github.io/) в браузере
2. Нажмите **Kate Mobile**
3. Авторизуйтесь и скопируйте токен из фрагмента URL
4. Вставьте в `.env` как `VK_TOKEN`

## Получение куков TikTok

1. Откройте tiktok.com в браузере и авторизуйтесь
2. Используйте расширение [cookies-txt](https://github.com/hrdl-github/cookies-txt) (или любой экспортер Netscape cookies)
3. Экспортируйте куки в `cookies.txt` в корень проекта

## Установка

```bash
cd tiktok-to-vk
pip install .
```

Или с [uv](https://github.com/astral-sh/uv):

```bash

```

## Настройка

```bash
cp .env.example .env
```

Отредактируйте `.env`:

| Переменная | Описание |
|---|---|
| `TIKTOK_COLLECTION_URL` | Ссылка на коллекцию TikTok |
| `TIKTOK_COOKIES_FILE` | Путь к cookies.txt (по умолчанию: `cookies.txt`) |
| `TIKTOK_PROXY` | Прокси для запросов к TikTok (опционально, рекомендовано для пользователей из России) |
| `VK_TOKEN` | Токен VK с правом загрузки видео |
| `APP_DB_PATH` | Путь к SQLite базе (по умолчанию: `data/clips.db`) |
| `APP_CHECK_INTERVAL` | Секунды между циклами в режиме демона (по умолчанию: `1800`) |
| `APP_ROTATE_VIDEO` | `True` = повернуть 90°, `False` = вписать с чёрными полосами |

См. `.env.example` для полного списка опций.

### Несколько коллекций

Укажите несколько ссылок через запятую в `.env`:

```
TIKTOK_COLLECTION_URL=https://www.tiktok.com/@user/collection/name-123,https://www.tiktok.com/@user/collection/other-456
```

## Использование

Одиночный цикл:

```bash
python main.py --once
```

Режим демона:

```bash
python main.py
```

### Docker

```bash
docker build -t tiktok-to-vk . --no-cache
docker run --env-file .env -e TIKTOK_COOKIES_FILE="/app/cookies.txt" -v ./cookies.txt:/app/cookies.txt -v ./data:/app/data tiktok-to-vk -d --restart=always
```


## Как это работает

1. **Обнаружение** -- вызов API `collection/item_list` для получения постов из коллекции
2. **Скачивание** -- загрузка видео через yt-dlp
3. **Обработка** -- конвертация горизонтальных видео в вертикальный 1080x1920
4. **Загрузка** -- отправка в VK Clips через VK API
5. **Очистка** -- удаление локальных файлов после успешной загрузки

Каждому видео добавляется описание:
- `Author:` -- имя автора в TikTok
- `Original:` -- ссылка на оригинальное видео в TikTok
- `Source:` -- ссылка на этот репозиторий

## Ограничения

- Фото-посты и карусели **не поддерживаются** в текущей версии
- Поддержка нескольких коллекций экспериментальна

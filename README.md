# tiktok-to-vk

Скачивает видео из коллекций TikTok и загружает их как VK Clips.

Скачивает видео из указанной коллекции (папка "Избранное") TikTok и загружает их на VK Clips от имени пользователя.
Ведет учет скачанных и опубликованных видео в локальной базе данных SQLite.
Работает в фоне как демон-программа на Python.
В случае простоя обрабатывает пропущенные видео.
Для каждого загруженного в VK видео в описании оставляет ссылку на оригинальное видео из TikTok, ссылку на TikTok пользователя чья коллекция и ссылку на GitHub репозиторий проекта.

Обрабатывает горизонтальные видео автоматически. Если видео горизонтальное, оно вписывается в формат 1080x1920 с черными полосами или поворачивается на 90 градусов. Режим настраивается через переменную APP_ROTATE_VIDEO в .env.

Обрабатывает фото-посты и карусели. Скачивает картинки и аудио, собирает в видео через pyav.

Для загрузки используется токен VK. Токен должен иметь право на загрузку видео.

## Установка

uv sync

## Настройка

Скопируй .env.example в .env и заполни:

TIKTOK_COLLECTION_URL - ссылка на коллекцию в TikTok
TIKTOK_COOKIES_FILE - путь к cookies.txt для доступа к TikTok
TIKTOK_PROXY - прокси для TikTok (опционально)
VK_TOKEN - токен VK
APP_DB_PATH - путь к базе данных
APP_CHECK_INTERVAL - интервал проверки в секундах
APP_ROTATE_VIDEO - True для поворота, False для вписывания (по умолчанию)

## Запуск

python main.py --once - одиночный цикл
python main.py - демон режим
